环境： 

- jdk1.8
- springboot 2.1.6 （MVC框架）
- mybatis（持久层）
- mysql（关系型数据库）
- druid（数据源监控台）
- idea（开发IDE）
- redis 3.0.0（缓存）
- ucloud-ufile(图片云服务)
- devtools(开发热部署)
- okhttp（模拟http请求）
